Phishing URL Detection API

This API provides phishing detection for URLs.

Endpoints

1. Predict a Single URL

                POST /predict
                Content-Type: application/json
                
                {
                    "url": "https://example.com"
                }

2. Batch Prediction

                POST /batch
                Content-Type: application/json
                
                {
                    "urls": [
                        "https://example1.com",
                        "https://example2.com"
                    ]
                }

3. Submit Feedback

                POST /feedback
                Content-Type: application/json
                
                {
                    "url": "https://example.com",
                    "actual_label": 1,
                    "predicted_label": 0
                }

4. Health Check

                GET /health
